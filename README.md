# Computational-Electromagnetism-FEM-Work-on-capacitors-waveguide-modes-and-scatterers
This work is about the implementation of a FEM code in Matlab in order to analyze structures such as capacitors, waveguides and scatterers. 
The work includes also comparisons between different methods, capacitance calculation via FEM, tracking of eigenvalues for waveguide modes via FEM, figures for electrical field, checking the limits of the method, results for the special case for FEM in scattering etc. 
This work was an assignment for the course of Computational Electromagnetism at the Aristotle University of Thessaloniki by Dimitrios Chatzidimitriou.
